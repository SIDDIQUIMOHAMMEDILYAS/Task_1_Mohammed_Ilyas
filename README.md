## üé® Customization

This project is designed to be easily customized.

### Changing Colors
Edit the `:root` variables at the top of the `<style>` section:
```css
:root {
    --primary-accent: #8b5cf6; /* Change this to your favorite color */
    --secondary-accent: #ec4899; 
    /* ... */
}
```

### Changing Content
Simply edit the text inside the HTML tags.
*   Update the `<title>` tag for browser tab names.
*   Replace the placeholder image URL (`picsum.photos`) in the `<img>` tag with a local file path (e.g., `images/profile.jpg`).

### Changing Name
Replace all instances of "Mohammed Ilyas Siddiqui" with your own name (Ctrl+F shortcut).
